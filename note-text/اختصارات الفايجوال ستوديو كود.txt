اختصارات فايجوال ستوديو كود :
 -CTRL + لفتح الاعدادات

-alt + سهم لفوق او لتحت بيحرك السطر يلي انت واقف عندو لفوق او لتحت

-shift +alt +لتحت او لفوق : بيعمل كوبي ولصق للسطر يلي واقف عندو

-ctrl +shift +k : بيمسح السطر

-ctrl + z : بيعمل الرجوع خطوة

-ctrl + enter : بينزل سطر لتحت

- ctrl + shift + enter : سطر جديد لفوق 

- ctrl + sheft + \ : بخليك تروح لقفل القوس المفتوح

-ctrl + [ او ] : بخليك تقرب السطر للامام او للخلف عدة خطوات

- home او end : بخليك تروح لاول السطر او لاخرو

-ctrl + home او end : بخليك تروح على اول الملف او اخرو 

-ctrl + / : بيعمل تعليق 

- sheft + alt + A : بيمعمل بلوك تعليق

-ctrl + sheft + M  : بيظهر المشاكل بالصفحة

- f8 او sheft +f8 : بخليك تتنقل بين المشاكل عندك بالصفحة

- تحديد الكلمة كاملة مع tab او sheft tab : بخليك تنتقل لتكتب بين الايليمينت الموجودة عندك

- ال f12 مع وضع الماوس ضمن الايدي او الكلاس : بتخليك تروح على تعديلات الايلمينت ضمن ملف ال css 
 
- alt + f12 : بيفتحها ضمن نافذة 

 - ctrl + h : بخليك تعمل تحديد لجميع الاسماء المتشابهة للتعديل عليها مع بعضها
